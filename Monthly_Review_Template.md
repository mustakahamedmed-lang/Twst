## 🏆 মোট পয়েন্ট অর্জন (এই মাসে)

```dataview
TABLE WITHOUT ID
  sum(rows.daily_points) AS "মোট পয়েন্ট",
  length(rows) AS "সক্রিয় দিন",
  round(average(rows.daily_points)) AS "দৈনিক গড়",
  round(average(rows.productivity_level), 1) AS "গড় প্রোডাক্টিভিটি লেভেল",
  round(average(rows.water_intake), 1) AS "গড় পানি (গ্লাস/দিন)"
FROM "01_Daily_Action_Plans"
WHERE file.ctime >= date(today) - dur(30 days)
GROUP BY "মাসিক সারাংশ"
```
